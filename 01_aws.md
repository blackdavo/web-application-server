# AWS에 대해서

#### AWS EC2에 대해서

* Elastic Compute Cloud(EC2)
	- 특징
		+ 한대의 컴퓨터를 임대해 주는 것과 같다
		+ 선호하는 운영체제 설치 및 웹서비스 프로그램 설치 가능
		+ 인터넷을 통해 클라우드 컴퓨터에 접속 할 수 있는 URL을 제공
		
	- EC2의 자원
		+ CPU
		+ 메모리
		+ 스토리지
		+ 네트워킹 용량
	
	- 기존 PC와으 차이점
		+ 인터넷으로만 접속 가능
		+ 구입비가 없고 사용한 만큼만 지불하면 됨
		+ 상황에 따라 변화된 환경을 상태 그대로 저장 가능 (이미지)
